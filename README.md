# Weather-App
A responsive Weather Web Application built using HTML, CSS, and JavaScript that fetches real-time weather data using OpenWeather API.
# 🌦 Weather App

A simple and responsive **Weather Forecast Web Application** that allows users to check real-time weather information of any city using live weather data.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* ☁️ Weather condition details
* 💧 Humidity information
* 🌬️ Wind speed display
* 📱 Responsive user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API

## 🔑 API Configuration

1. Create a free account at https://openweathermap.org
2. Generate your API Key
3. Replace API key inside `script.js`

```javascript
const apiKey = "YOUR_API_KEY";
```
